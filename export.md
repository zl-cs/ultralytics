# Export .onnx to .plan (NX-1)
- Prepare model folder

    ```
    mkdir pretrained_model
    cd pretrained_model
    mkdir plan_model
    mkdir onnx_model
    ```
- Copy [onnx models generated by lion-11 machine](https://github.com/zl-cs/ultralytics/blob/lion-11/export.md) to "onnx_model" folder
- Export command
    ```
    bash export_plan.sh
    ```
